# Hash Map
implementation of the hash map with double hashing algorithm
## Prerequisites
* **JDK >= 1.8.0_131**;
* **Apache Maven >= 3.5.2**
## Project structure 
* src\main\java\com.peterholub.hashmap.HashMap.java - Hash Map implementation of Map interface 
## Run Tests
Run script for Test
```
mvn test
```
